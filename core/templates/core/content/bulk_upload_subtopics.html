JSON Format Guide

Important: All fields are required

Each subtopic must reference a valid exam board, subject, grade, and topic combination. The parent topic must have a grade assigned.

Required Fields:

  • exam_board - Exam board abbreviation (e.g., "ZIMSEC", "CIE")
  • subject - Subject name (e.g., "Biology")
  • grade - Grade number (e.g., 10)
  • topic - Parent topic name (e.g., "Cell Biology")
  • name - Subtopic name

Optional Fields:

  • description - Brief description
  • order - Display order (default: 0)
  • is_active - true/false (default: true)
